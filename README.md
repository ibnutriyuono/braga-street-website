# Web Development Batch 3

* [Bootstrap](http://http://getbootstrap.com/) - The web framework used :+1:

* [Traversy Media](https://www.youtube.com/channel/UC29ju8bIPH5as8OGnQzwJyA) - Tutorial how to create a firebase email form
